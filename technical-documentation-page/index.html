Back End

La seguente è una lista dei linguaggi di programmazione utilizzati per la creazione dell'app.

  • HTML, per il disegno delle pagine web
  • CSS, per la silizzazione delle pagine webm edesime, scritte in HTML
  • Javascript, per le funzioni di inerattività, fra cui l'inserimento dei dati, delle pagine.
  • mySQL, per la raccolta e la gesione degli ordini sotto forma di database, ossia di tabella ordinata a più parametri.
  • Python, per l'analisi statistica degli ordini a posteriori.
